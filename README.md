# lrtest-web

基于vue-admin-template模板开发的一个测试工具平台（前端）

项目演示传送门：https://lrtest.chdxia.com

Jenkins传送门：https://jenkins.chdxia.com

后端传送门：https://github.com/chdxia/lrtest-api

### 开发

```bash
# 安装依赖
npm install

# 启动服务
npm run dev
```
### 发布

```bash
# 生产环境打包
npm run build:prod
```

参考：https://github.com/PanJiaChen/vue-element-admin
